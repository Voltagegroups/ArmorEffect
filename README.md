### Getting the config.yml of the plugin

```YAML
Iron Helmet: #name of item Armor
  message: "message" # or null
  effect:
    1:
      durability: 2 #time in seconde
      visible: true # or false particules
    3:
      durability: 2 #time in seconde
      visible: true # or false particules
```
